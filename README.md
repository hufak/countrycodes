# Country codes

A handy CSV file mapping the arcane Austrian university student registry "country" codes to their corresponding (English) names, linguistic/political region, continent, and their nationals' tuition fee status (according to Austrian university law and university-specific refund regulations).

Only contains 87 codes, so the list is definitely incomplete!

You can use raw download from the following URL: https://hufak.github.io/countrycodes/countries.csv
